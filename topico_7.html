Margens CSS

As marginpropriedades CSS são usadas para criar espaço em torno de elementos, fora de qualquer borda definida.

Com CSS, você tem controle total sobre as margens. Existem propriedades para definir a margem para cada lado de um elemento (superior, direita, inferior e esquerda).

Margem - Lados Individuais

CSS tem propriedades para especificar a margem para cada lado de um elemento:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Todas as propriedades de margem podem ter os seguintes valores:

  • auto - o navegador calcula a margem
  • length - especifica uma margem em px, pt, cm, etc.
  • % - especifica uma margem em% da largura do elemento contido
  • inherit - especifica que a margem deve ser herdada do elemento pai

Dica:valores negativos são permitidos.

Margem - Propriedade de taquigrafia

Para encurtar o código, é possível especificar todas as propriedades da margem em uma propriedade.

A marginpropriedade é uma propriedade abreviada para as seguintes propriedades de margem individuais:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Então, aqui está como funciona:

  • margem: 25 px 50 px 75 px x 100 px;
    • a margem superior é 25px
    • margem direita é 50px
    • margem inferior é 75px
    • margem esquerda é 100px

Exemplo

Se a marginpropriedade tiver três valores:

  • margem: 25 px 50 px 75 px;
    • a margem superior é 25px
    • margens direita e esquerda são 50px
    • margem inferior é 75px

Exemplo

Se a marginpropriedade tiver dois valores:

  • margem: 25 px 50 px;
    • as margens superior e inferior são de 25px
    • margens direita e esquerda são 50px

Exemplo

Se a marginpropriedade tiver um valor:

  • margem: 25px;
    • todas as quatro margens são 25px

Colapso da margem

As margens superior e inferior dos elementos são às vezes recolhidas em uma única margem que é igual à maior das duas margens.

Isso não acontece nas margens esquerda e direita! Apenas margens superior e inferior!
